# Quiz Generator

A modern **AI-powered Quiz Generator** built using **HTML, CSS, and JavaScript**.  
Generate multiple-choice questions (MCQs) based on any topic and difficulty level.

## Features

- Enter any **topic** and select **number of questions**.
- Choose **difficulty**: Easy, Medium, or Hard.
- Polished and responsive **UI**.
- Demo mode available for safe GitHub Pages deployment.
